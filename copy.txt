

All - This is the first milestone release for the Nexus Suite Tech Live project, covering the following core functionality:

Nexus REST API for on-boarding of Citi Apps
Nexus Async API for Signer Service
MQ Messaging using RH AMQ
Support for bulk requests for Signer service
Phased validations of request messages and acks
Signer adapter for nCipher service (Phase-1)
AMQP support for MQ messaging
Support for positive and negative testcases

What's not covered in this release:
nCipher Phase-2 API
REST API for Nexus Router functionality
DLT Adapters

We have scheduled a demo of the functionality this morning.
